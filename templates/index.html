Chào mừng bạn đến với AegisBot Orchestrator

Hệ thống máy chủ quản lý đa tiến trình chuyên dụng, tối ưu hóa đặc biệt cho VPS cấu hình nhẹ. AegisBot tự động cô lập môi trường của từng bot và tự động sửa lỗi thiếu thư viện runtime mà không làm gián đoạn hệ thống.

Chống sập (Resiliency)

Nếu bot bị crash do lỗi code hoặc mất kết nối mạng, AegisBot sẽ tự động khởi động lại sau 5 giây.

Tự động sửa lỗi thiếu thư viện

Tự động quét code nguồn bằng AST và cài đặt gói PyPI còn thiếu. Nếu phát hiện lỗi ModuleNotFound lúc chạy, hệ thống sẽ tự tải và restart.

Tối ưu hóa RAM VPS 1GB

Hàng đợi cài đặt Pip chạy tuần tự giúp máy chủ không bao giờ bị quá tải RAM khi cài thư viện.

Hướng dẫn nhanh

  1. Bấm nút "Thêm Bot" ở góc trái.
  2. Đặt ID (ví dụ: my_bot), dán mã nguồn Python của bot và thiết lập cấu hình .env (chứa TOKEN của bot).
  3. Bấm "Lưu Bot". Hệ thống sẽ tự tạo Virtual Environment và cài đặt các thư viện cần thiết.
  4. Bấm "Chạy" và theo dõi log hoạt động thời gian thực tại màn hình Console.
